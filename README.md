# 📚 Sistema de Gestión de Biblioteca 

Un sistema completo para administrar libros, préstamos y búsquedas en una biblioteca, desarrollado en **C# (.NET)** con persistencia en JSON.

## 🚀 Características principales
- **Gestión de libros**: Añadir, editar y eliminar libros (ISBN, título, autor, género, cantidad).
- **Préstamos**: Registro de préstamos con fecha y usuario.
- **Búsquedas inteligentes**: Filtrado por título, autor o género.
- **Persistencia**: Datos guardados en archivos JSON (`libros.json` y `préstamos.json`).
- **Interfaz CLI**: Menú interactivo con validación de entradas.
